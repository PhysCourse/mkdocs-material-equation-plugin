# CourseTutorials

В этом файле будет рассказано как все установить и оформлять.

## Установка

- Установка пакета mkdocs и расширения material

```bash
    pip install mkdocs
    pip install mkdocs-material
```

## Структура проекта

```text
Book/
├──docs/ # Это корень: все файлы проекты индексируются отсюда
|  ├──Chapter1/
|  |  ├──P1.md
|  |  ├──P2.md
|  |  └──P3.md
|  ├──css/ # опционально, если захочется менять стили
|  ├──javascripts/
|  |  └──mathjax.js # пригодится для подключения математики
└──mkdocs.yml # файл со всей конфигурацией: индексация страниц, плагины и т.д. 
```

- Для сборки нужно запустить команду

```bash
    mkdocs build
```

в папке, где находится файл ```mkdocs.yml```

- Для того, чтобы сайт изменялся "на лету" (при изменениях), нужно запустить команду

```bash
    
```

и открыть в браузере сайт ```localhost:8000``` (по умолчанию порт 8000, но можно изменить на любой).
